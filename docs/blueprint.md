# **App Name**: HireHub

## Core Features:

- Secure User Authentication: Enable secure registration and login for both job seekers and recruiters using JWT. Implement role-based access control.
- Job Seeker Profile Management: Allow job seekers to create, view, and update their profiles, including skills, work experience, and a resume link.
- Job Discovery & Search: Provide a paginated list of all available jobs, with robust search and filtering capabilities by job title, location, and skills.
- Job Application & Tracking: Allow job seekers to apply for jobs directly and view the history and status of their submitted applications.
- Recruiter Job Management: Empower recruiters to create, edit, and delete their job postings, and manage the job status (Open/Closed).
- Applicant Overview for Recruiters: Recruiters can view a comprehensive list of all applicants for their posted jobs.
- AI Job Description Keyword Tool: An AI tool for recruiters to suggest relevant keywords and skills based on the job title and description, optimizing job posting visibility.

## Style Guidelines:

- Color Anchor: Professional efficiency. The chosen scheme is light, prioritizing readability and a clean corporate feel.
- Primary color: A deep muted indigo (#363673), evoking stability, trust, and professionalism, and providing strong contrast against a light background.
- Background color: A very light indigo tint (#F6F6FA), offering a clean, expansive canvas that complements the primary color.
- Accent color: A brighter, classic blue (#3399CC), selected as an analogous hue to the primary. This provides a clear highlight for interactive elements and calls to action.
- Headline and body font: 'Inter', a grotesque-style sans-serif. Its modern, objective, and neutral aesthetic ensures excellent readability and a polished, professional look suitable for all content.
- Utilize modern, minimalist line icons that ensure clarity and immediate recognition, contributing to a clean and efficient user experience.
- A clean, spacious layout characterized by ample whitespace, a flexible single-column content flow, and responsive card-based designs for listings to maintain readability across devices.
- Incorporate subtle, functional animations for UI transitions (e.g., loading states, filtering results, page changes) to provide fluid feedback without distracting from the core experience.
